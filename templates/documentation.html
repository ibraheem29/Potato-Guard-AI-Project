info Overview

Potato Pulse is an AI-powered web application that detects diseases in potato plants using deep learning. The system analyzes potato leaf images and classifies them into three categories: Healthy, Early Blight, and Late Blight.

help How to Use

Using the Web Application

  1. Navigate to the home page
  2. Click on the upload area or drag and drop a potato leaf image
  3. Supported formats: JPG, PNG, JPEG
  4. Click "Analyze Image" to get the prediction
  5. View the disease classification and confidence score

computer Running Locally

Option 1: Using Docker (Recommended)

Pull and run the Docker image:

# Pull the image from Docker Hub docker pull salman5243/potato-disease-detection:latest # Run the container docker run -d -p 8080:8080 --name potato-app salman5243/potato-disease-detection:latest # Access the application at http://localhost:8080

Option 2: Using Docker Compose

Create a docker-compose.yml file:

version: '3.8' services: potato-disease-app: image: salman5243/potato-disease-detection:latest container_name: potato-disease-detection ports: - "8080:8080" restart: unless-stopped

Then run:

docker-compose up -d

Option 3: From Source Code

Clone the repository and run locally:

# Clone the repository git clone https://github.com/Salman5242/Potato-Disease-Detection-MLops.git cd Potato-Disease-Detection-MLops # Install dependencies pip install -r requirements.txt # Run the application python app.py # Access at http://localhost:8080

model_training Model Information

The AI model is a Convolutional Neural Network (CNN) trained on the PlantVillage dataset. It can classify potato leaves into three categories:

  • Healthy: No disease detected
  • Early Blight: Caused by Alternaria solani
  • Late Blight: Caused by Phytophthora infestans

api API Endpoint

For programmatic access, use the testing endpoint:

POST /testing Content-Type: multipart/form-data Parameters: - file: Image file (jpg, png, jpeg) Response: { "prediction_class": "Potato___healthy", "confidence": 95.23 }

Example using cURL:

curl -X POST http://localhost:8080/testing \ -F "file=@potato_leaf.jpg"

troubleshoot Troubleshooting

Port Already in Use

If port 8080 is already in use, map to a different port:

docker run -d -p 8081:8080 --name potato-app salman5243/potato-disease-detection:latest

Model Not Found

Ensure the model file exists at: Models/.keras/Alpha.keras

Low Accuracy Predictions

  • Use clear, well-lit images of potato leaves
  • Ensure the leaf fills most of the image frame
  • Avoid blurry or low-resolution images
